# Inteligência Artificial aplicada ao desenvolvimento de Bots com Python

## 🎯 Contexto e Objetivos

> Este projeto foi desenvolvido como parte do desafio prático da DIO com o objetivo de explorar o NotebookLM como ferramenta de aprendizagem ativa.

O tema escolhido foi Inteligência Artificial aplicada ao desenvolvimento de bots e automações utilizando Python, por estar alinhado com meus estudos atuais em programação e com os projetos pessoais que estou desenvolvendo.

Objetivos de estudo:
Compreender como a IA pode auxiliar no desenvolvimento de bots;
Aprender boas práticas na criação de prompts;
Organizar conteúdos técnicos em um ambiente inteligente de estudos;
Criar um material reutilizável para futuras revisões.

## 🤖 Engenharia de Prompts e Cicatrizes

> Prompt 1

"Explique como a IA pode auxiliar no desenvolvimento de bots em Python para iniciantes."

Resultado:

Obtive uma visão geral clara sobre automação, geração de código e suporte à documentação.

Dificuldade encontrada:

As respostas eram muito amplas.

Ajuste realizado:

"Explique utilizando exemplos relacionados a bots financeiros desenvolvidos com Python."

Prompt 2

"Quais são as principais bibliotecas Python utilizadas para construção de bots?"

Resultado:

Foram sugeridas bibliotecas como:

python-telegram-bot;
discord.py;
FastAPI;
Django.
Dificuldade encontrada:

Pouca comparação entre as opções.

Ajuste realizado:

"Compare vantagens e desvantagens das bibliotecas para projetos pequenos e grandes."

## 📖 Miniguia de Estudo

### Resumos Estruturados

O que é IA Generativa?

Tecnologia capaz de criar conteúdos novos com base em padrões aprendidos a partir de grandes volumes de dados.

Como a IA auxilia programadores?
Explicando conceitos;
Revisando códigos;
Gerando exemplos;
Sugerindo melhorias;
Auxiliando na documentação.
Aplicações em Bots
Atendimento automatizado;
Gestão financeira;
Envio de notificações;
Integração com APIs;
Organização de informações.

### Glossário

> API: Interface que permite comunicação entre sistemas.

Bot: Programa capaz de executar tarefas automaticamente.

Django: Framework web desenvolvido em Python.

Prompt: Instrução fornecida à IA para obtenção de uma resposta.

NotebookLM: Ferramenta do Google que utiliza IA para organizar e interagir com materiais de estudo.

IA Generativa: Modelo de inteligência artificial capaz de criar novos conteúdos.
